Comment ça marche ?

1

Publiez votre site

Créez un compte gratuit, remplissez les informations de votre site (captures d'écran, description, prix, technologies utilisées) et mettez-le en ligne en quelques minutes.

2

Découvrez et choisissez

Les acheteurs parcourent les sites disponibles avec des filtres (catégorie, prix, type de site). Ils peuvent voir les détails et contacter le vendeur.

3

Transaction sécurisée

Une fois l'accord conclu, le paiement est sécurisé via la plateforme. Fortuna protège à la fois l'acheteur et le vendeur.

4

Transfert du site

Vous transférez les accès (hébergement, domaine, codes sources). Nous vous accompagnons pour que tout se passe bien.
